**ChatYok** es una plataforma de mensajería integral diseñada para ofrecer una experiencia fluida. Cuenta con aplicaciones móviles nativas, una interfaz web dinámica y un backend robusto para mantener a los usuarios siempre conectados de forma segura.

### 🔧 Tecnologías utilizadas

<p align="center">
  <img src="https://shields.io" alt="React Native" />
  <img src="https://shields.io" alt="Next.js" />
  <img src="https://shields.io" alt="Node.js" />
  <img src="https://shields.io" alt="Socket.IO" />
  <img src="https://shields.io" alt="MongoDB" />
  <img src="https://shields.io" alt="JWT" />
</p>

* **Frontend móvil:** React Native *(Android e iOS)*
* **Web y API:** Next.js *(framework React)*
* **Servidor backend:** Node.js con Express.js
* **Comunicación en tiempo real:** Socket.IO
* **Autenticación:** JWT (JSON Web Token)
* **Base de datos:** MongoDB o base de datos NoSQL compatible

---

### ✨ Características

* 💬 **Chat individual** en tiempo real.
* 🔐 **Registro e inicio de sesión** de usuario seguros con JWT.
* 🟢 **Indicadores de estado** en línea / fuera de línea.
* ✍️ **Indicadores de escritura** y confirmaciones de lectura.
* 🔔 **Notificaciones** de nuevos mensajes.
* 📷 Envía **texto, imágenes y emojis**.
* 📱💻 Interfaz de usuario **limpia, moderna y adaptable** (móvil y web).

---

### 📱 Módulos Principales

#### 📱 Aplicación móvil
Desarrollada con **React Native**, brindando un rendimiento fluido y una experiencia similar a la nativa tanto en plataformas **Android** como **iOS**.

#### 🌐 Aplicación web
Impulsada por **Next.js**, optimizada con renderizado del lado del servidor (SSR) y un diseño totalmente adaptable para cualquier navegador.

---

### 🔒 Seguridad

* 🛡️ Autenticación robusta basada en tokens con **JWT**.
* 🔑 Contraseñas almacenadas de forma segura mediante **hashing**.

### ✨ Características principales

* 💬 **Chat individual en tiempo real**
  * Envía y recibe mensajes al instante. 
  * Actualizaciones en tiempo real gracias a Socket.IO.

* 🔐 **Autenticación segura**
  * Inicio de sesión y registro de usuarios.
  * Tokens JWT para un acceso seguro y fiable.

* 👤 **Presencia del usuario (en línea/fuera de línea)**
  * Estado de conexión visible.
  * Vea cuándo otros usuarios están activos en tiempo real.

* ✍️ **Indicador de escritura**
  * Estado dinámico en el chat.
  * Se muestra cuando el otro usuario está escribiendo un mensaje.

* ✅ **Confirmaciones de lectura**
  * Seguimiento de entrega de mensajes.
  * Sepa con certeza cuándo se han leído sus mensajes.

* 🔔 **Notificaciones de mensajes**
  * Alertas instantáneas.
  * Reciba notificaciones dentro de la aplicación o notificaciones push.

* 📷 **Mensajería con imágenes**
  * Comparta archivos visuales.
  * Envíe fotos directamente en el chat.

* 🖥️📱 **Interfaz de usuario multiplataforma**
  * Diseño limpio, adaptable y moderno.
  * Experiencia homogénea tanto para web (Next.js) como para móviles (React Native).
