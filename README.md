# Grovix 🛡️

**Grovix** es una aplicación de chat multiplataforma (Web y Móviles) actualmente **en desarrollo**, diseñada bajo la identidad visual de **MD Studios**. El proyecto cuenta con un simulador interactivo completo basado en una interfaz premium color carbón y blanco, con navegación fluida y un formato de chat limpio sin burbujas.

## Características Principales 🚀

- **Interfaz Premium:** Estética sofisticada en tonos carbón oscuro y blanco ultra-premium, optimizada para pantallas web y móviles.
- **Chat Limpio:** Sistema de mensajería continuo en formato compacto (tipo IRC/Discord) sin burbujas de texto tradicionales.
- **Sistema de Clanes:** Creación y gestión de clanes con generación automática de enlaces únicos de invitación (`grovix.app/invite/ID`).
- **Estados de Actividad:** Soporte persistente para estados de disponibilidad (En línea, Ausente, No molestar) con indicadores visuales de color.
- **Soporte Multi-idioma:** Localización completa integrada con cambio dinámico entre **Español** e **Inglés**.
- **Perfil de Usuario:** Tarjetas de perfil personalizadas basadas en el almacenamiento local para la persistencia de datos.
