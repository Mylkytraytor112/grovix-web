**ChatYork**

Esta es una aplicación de chat en tiempo real desarrollada con React Native para dispositivos móviles (Android e iOS), Next.js para la interfaz web y la API, y Node.js para el backend. La aplicación ofrece una experiencia de mensajería fluida y con gran capacidad de respuesta, con comunicación en tiempo real y autenticación segura.

🔧 Tecnologías utilizadas:
Desarrollo frontend para móviles: React Native (Android e iOS)
Web y API: Next.js (framework React)
Servidor backend: Node.js con Express.js
Comunicación en tiempo real: Socket.IO
Autenticación: JWT (JSON Web Token)
Base de datos: MongoDB o base de datos NoSQL compatible.

✨ Características:
Chat individual en tiempo real
Registro e inicio de sesión de usuario con JWT
Indicadores de estado en línea/fuera de línea
Indicadores de escritura y confirmaciones de lectura
Notificaciones de nuevos mensajes
Envía texto, imágenes y emojis.
Interfaz de usuario limpia, moderna y adaptable (móvil y web).
📱 Aplicación móvil:
React Native ofrece un rendimiento similar al nativo tanto en plataformas Android como iOS.

🌐 Aplicación web:
Next.js impulsa la versión web con renderizado del lado del servidor y un diseño totalmente adaptable.

🔒 Seguridad:
Autenticación basada en tokens con JWT
Contraseñas almacenadas de forma segura mediante hash.
